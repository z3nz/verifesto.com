# Verifesto Studios

Creative development studio that turns bold visions into reality.

## Stack

- **Frontend:** Vue 3 + Vite + Tailwind CSS → Cloudflare Pages
- **Backend:** Django + DRF + PostgreSQL → Railway

## Development

```bash
# Frontend
cd frontend && npm install && npm run dev

# Backend
cd backend && pip install -r requirements.txt
python manage.py migrate && python manage.py runserver
```

## Deployment

- Frontend: Cloudflare Pages (auto-deploy from main)
- Backend: Railway (auto-deploy from main)
- Domain: verifesto.com / api.verifesto.com
